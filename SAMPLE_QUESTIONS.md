Question 1: What does the Turing Test, also known as the imitation game, evaluate? 
A: Whether a machine can solve complex mathematical problems faster than humans Feedback: Incorrect. Speed or calculation ability is not the focus of the Turing Test.
B: Whether a machine can accurately store and retrieve large volumes of information Feedback: Incorrect. The test is not about memory or data storage capabilities.
 *C: Whether a machine's responses are indistinguishable from a human's in conversation Feedback: Correct! The Turing Test assesses if a machine can imitate human-like intelligence through interaction. 
D: Whether a machine can learn patterns from data without human supervision Feedback: Incorrect. This describes unsupervised learning, not the Turing Test.

Question 3: What best describes Deep Learning (DL)? 
A: A rule-based approach that relies on manually programmed logic Feedback: Incorrect. Deep Learning does not depend on predefined rules; it learns from data. 
*B: A branch of Machine Learning that uses multi-layered neural networks inspired by the human brain Feedback: Correct! Deep Learning models use many stacked layers to learn complex patterns from data. 
C: A data storage technique designed to handle very large datasets efficiently Feedback: Incorrect. While DL uses large datasets, it is not a data storage method. 
D: A simple statistical method that works only with small, structured datasets Feedback: Incorrect. Deep Learning is designed for complex tasks and often works with unstructured data.
