# 🧩 KC Builder 01: Knowledge Check Generator

> **The ultimate bridge between raw content and Articulate Rise 360.** Convert hundreds of Multiple Choice (MCQ) and Multiple-Response (MMCQ) questions into a beautiful, interactive Knowledge Check block in seconds. No coding, no manual entry into Rise, and no SCORM headaches.

---

### 🚀 Key Features
* ⚡ **Bulk Parsing:** Paste 1 to 500+ questions at once.
* 🤖 **AI-Powered:** Built-in generators for Gemini, Claude, and GPT-4o.
* 🛠️ **White-Labeling:** Custom brand colors and logo presets.
* 🧪 **QA Sandbox:** Preview and test every question before exporting.
* 📂 **Rise 360 Ready:** One-click "Copy HTML" for the Rise Embed block.
* 🔄 **Custom Logic:** Unlimited attempts and per-question feedback.

---

### 📸 How it Works
1. **Paste:** Drop your questions using the simple `*` notation for correct answers.
2. **Review:** Use the **QA Preview** tab to verify logic and feedback.
3. **Export:** Click **"Copy HTML for Rise 360"**.
4. **Embed:** In Articulate Rise, add an **Multimedia > Embed** block and paste the code.
