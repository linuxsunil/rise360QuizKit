# Contributing to Rise 360 Quiz Kit

Thank you for your interest in improving this tool! As a **Performance-First** project, we prioritize changes that reduce friction for Instructional Designers and improve learner outcomes.

## How to Contribute

### 1. Reporting Bugs
* Check the current Issues tab to see if the bug has been reported.
* Provide a sample of the text that caused the parsing error.
* Mention your browser and Rise 360 version.

### 2. Suggesting "Action Mapping" Improvements
I am particularly interested in refining the AI prompts to move away from "information-dump" questions. If you have better framing for scenario-based prompts, please open a Discussion.

### 3. Code Contributions
* Keep it simple: This is a **single-file HTML/JS** tool to remain "non-techie" friendly.
* Please do not add heavy external frameworks or dependencies.
* Ensure all new features (like the **Granular QA Reset**) are fully tested in the QA Preview tab before submitting a Pull Request.

## Code of Conduct
Be kind and professional. We are building this to help the ID community grow.
